## Installation
### Installing dependencies
From within the Calculator directory:
```sh
npm install
```
### Creating the bundle file
From within the Calculator directory:
```sh
npm run build (development mode)
npm run prod (production mode)
```
### Starting the server
From within the Calculator directory:
```sh
npm start
```
### Running tests
```sh
npm test
```
